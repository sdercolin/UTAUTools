UST重设工具「USTRESET」 v1.1
作者：科林/sdercolin 
http://weibo.com/sdercolin 
sdercolin@outlook.jp

本程序可以通过简单的操作将UST文件的参数进行清空，并且预置一些必要的参数，方便调声。

※简易使用步骤
1.将程序复制到UTAU程序根目录下（与utau.exe放在同一文件夹下），运行程序；
2.点击“载入UST”，选择需要重设的UST；
3.点击“载入默认设置”（如果只需要进行少数调整，可以省略这一步）；
4.按需要进行必要的设置调整，如转换歌词种类，长休止符切割等；
【重要】如果您看不懂某些选项，请尽量遵照“默认设置”；
5.保存或另存为UST。

※如系统版本较低可能无法运行，请下载安装Microsoft .NET Framework 2.0或以上版本：http://www.microsoft.com/zh-cn/download/details.aspx?id=1639

※一些功能上的说明（新手可以略过）
1.UST属性设置中的“Flags”对应的是整个工程的Flags，亦即“生成時オプション”；
2.转换歌词会自动清除前缀和后缀；如果只需要清除而不转换也请勾选这一步，然后将原歌词种类和目标歌词种类均设置为当前的歌词种类； ※原歌词为罗马单独音的场合，如音符含有除“?”以外的前缀将会被忽略；
3.切割长休止符的参数使用了 遊牧家族 的“長い休符を分割”插件的预置参数；
4.假名和罗马音的相互转换使用了 ちえP 的 “ひらがな→ローマ字・ローマ字→ひらがな”插件的对应表；
5.重置Pitch的锚点个数：如果设置为2，对应UTAU中的“平均分布”不被勾选，选择3及以上则对应勾选“平均分布”；
6.默认Flags对应的是每个音符的Flags，默认值设置为Y0L5；Flags含义说明见：http://bbs.ivocaloid.com/thread-66570-1-1.html
7.默认BRE框只有在勾选默认Flags后才能使用，为空设置为50（范围0~100）；
8.颤音曲线的“预设”对应UTAU程序的预设颤音模型。
